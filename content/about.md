# About this project

This is intentionally small.

The goal is to have a maintainable codebase that still looks and behaves like a real project:

- content lives in `content/`
- application code lives in `src/`
- markdown is compiled into static HTML at build time
- the top navigation is derived from the content tree

It is a good baseline for demonstrating review comments on both code and editorial changes.
