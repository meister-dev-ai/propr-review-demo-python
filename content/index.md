# Review-ready demo content

This repository starts with a simple static blog built from markdown content.

- Root markdown files become top-level pages.
- Root directories with an `_index.md` file become top-level sections.
- Additional markdown files inside a section become article entries for that section.

That gives us enough structure to create realistic pull requests against content, layout, and build logic.
